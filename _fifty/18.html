--- number: 18 layout: fifty title: The Search description: Riffing on the spot effect from Glow 17, I decided to create an adaptation that would use the same visual effect for a very different meaning, transforming stage lights into search lights. Utilizing the same semi-transparent triangles, I implemented keyframe animations that moved the triangles around an origin point set at the tallest corner of eachtriangle. By moving each light in this way, rotating them around this particular point and creating a certain kind of cadence, they appear to be in pursuit, moving like the search lights in spy movies and police dramas to find a very special something. permalink: fifty/18/ ---
